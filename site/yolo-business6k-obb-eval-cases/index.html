指标说明

Precision 0.856

模型预测出来的 overlay 框里，有多少比例是对的。这个值越高，误检越少；当前说明预测框整体可信度还可以，但仍有一部分商品文字或画面文字被当成 overlay。

Recall 0.883

真实存在的 overlay 中，有多少被模型找到了。这个值越高，漏检越少；当前对有标注 overlay 的视频帧覆盖较好，134 张测试图中正样本没有完全无预测的 case。

mAP50 0.889

在 IoU 0.50 的匹配标准下综合衡量检测排序和命中情况。它更偏“是否大致框到目标”，当前分数较高，说明 YOLO 路线可以抓到主要 overlay 区域。

mAP50-95 0.556

在 IoU 0.50 到 0.95 多个严格阈值下的平均分。它更关注框的位置和角度精细度；当前明显低于 mAP50，说明边界贴合、四边形角点精度还有提升空间。

Clean FP 16/25

25 张无 overlay 的 clean 测试图中，有 16 张出现预测框。这个指标是当前主要风险，代表模型容易把商品原生文字、UI 或背景文字误判为可移除 overlay。

Positive no pred 0/109

109 张有 overlay 的测试图中，没有出现完全无预测的图片。它说明模型召回覆盖较好，但不能代表每个框都精确，也不能抵消 clean 误检问题。
