重要限制

本轮不是最终配音质量验收。 目标文本为 smoke-test 脚本，不是对源视频逐句 ASR + 翻译后的严格对齐文案；时长控制也尚未调参。

  • 可用于判断模型能否部署、是否能出声、语种发音是否值得继续。
  • 不应用于判断最终口型同步、完整翻译质量或线上生产稳定性。
  • JaiTTS 的参考 transcript 仍是近似值，正式评估要补 exact transcript。

风险与下一步

1. 先做母语者听评

本轮已经具备听评材料。建议按发音准确性、自然度、音色相似度、语调保留四项打分，优先确认 TH、VN、ID、MY、BR、TW。

2. 再做时长控制

目前大多数输出明显短于源视频。下一轮应使用 ASR 时间戳、逐句翻译、文本长度约束、语速参数和必要的 time-stretch 做对齐。

3. 补齐生产验证

需要补多人声切分、带 BGM 视频的人声分离、批量失败率、GPU 吞吐、授权与生成日志。通过后再决定是否接入 API 兜底。

本页面为 smoke test 汇报页，视频仅用于模型验证和内部听评。正式上线前需重新使用授权素材、准确翻译文本、逐句对齐、母语者听评和合规审查。
