数据集：work/ppocr_business_train_clean_synth_6k_v1，来源为 reviewed 业务帧 benchmark/adtext_benchmark_merged_reviewed.json，按 video/group 划分，val/test 视频不参与训练或合成。

指标口径

这里复现 PaddleOCR DetMetric 的核心口径：每个 GT 四边形和预测四边形计算 polygon IoU，按 IoU 从高到低做一对一贪心匹配；IoU ≥ 0.50 记为 TP，未匹配 GT 记为 FN，未匹配预测记为 FP。

Precision = TP / (TP + FP) Recall = TP / (TP + FN) Hmean = 2 * Precision * Recall / (Precision + Recall)

本页只评估 test_business_overlay.txt 中有 overlay 的正样本帧，不包含 clean/native 空样本误检统计；clean FP 需要单独跑负样本推理后按“有无预测框 / 每图预测框数 / 与 native 区域重叠”统计。
