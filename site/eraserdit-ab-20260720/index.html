50/3、30/3、30/1 参数对比

两条泰国短视频使用同一输入、同一 mask、同一 prompt，分别验证 num_inference_steps 与 guidance_scale 对完整推理耗时和擦除质量的影响。

当前建议：生产默认优先使用 30 / CFG 3。本次样本中它比 50 步更快，质量代理指标与 50/3 接近；30/1 可作为追求最低延迟的 fallback，但正式上线前仍建议对更多复杂 mask 做抽样复核。

指标定义与限制

  • mask 外 MAE / changed%：source 与 output 在 mask 外区域的平均绝对像素差、差异超过 8 灰度的像素比例，用于衡量非目标区域保持度。
  • mask 内 MAE：擦除区域的变化幅度，只说明模型确实改动了多少，不代表内容一定更真实。
  • 输出黑像素% / 黑块代理帧：mask 内接近黑色的像素比例，以及相对 source 明显增加黑像素的帧数，用于捕捉黑块风险。
  • 帧数差异：视频 2 的 OpenCV 解码结果是 source 413 帧、mask/output 414 帧；容器声明均为 414 帧。该差异对三组完全一致，报告不将其归因于参数。
