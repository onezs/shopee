建议 Next Step

1. 保持弱拦截策略

video_url 为空、duration 为空可以硬拦截；duration <= 10、duration >= 40 建议先做质量标签或降权，避免一次性损失 14.09% 商品覆盖。

2. 优选实验聚焦多候选 item

只有 15.28% 的 item 有 2-3 条候选视频，排序实验应优先在这部分上评估收益；单候选 item 主要判断是否保留或补召回。

3. 分站点单独定阈值

ID 的基础拦截损失低于 SG，但候选池仍薄。后续不建议直接复用同一套硬阈值，应按站点评估覆盖和业务收益。
