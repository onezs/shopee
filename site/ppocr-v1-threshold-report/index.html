Recommendation

建议：线上候选仍先用 V1 continue，推荐工作点 box_thresh=0.40。相比官方预训练，V1 在同口径下把 precision 从 47.66% 提到 81.03%，clean FP 从 11/25 降到 4/25，同时 recall 从 91.50% 提到 94.00%。V7 best 已修复 V6 退化但仍未超过 V1；下一轮提升重点继续围绕细长/低对比 watermark，同时避免牺牲 title/subtitle recall 和 clean FP。
