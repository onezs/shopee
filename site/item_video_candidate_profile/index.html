L4 策略建议

1. 统一使用 CP 视频底表口径

SG 和 ID 均使用 `dim_cp__video_info__df` 作为候选池来源。后续视频优选、质量标签和覆盖评估建议统一沿用这一口径，避免不同口径之间混用。

2. 拦截规则先做标签化

直接硬拦截会让 SG 21.38%、ID 11.96% 商品无视频。建议先将空封面、过短、过长做质量标签或降权，再看业务覆盖和素材质量收益。

3. 优选分层推进

ID 候选池更厚，可优先做 topN 排序实验；SG 单候选占比更高，需要更谨慎处理硬拦截和补召回，避免误伤覆盖。
